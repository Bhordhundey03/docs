---
title: "Create dispute"
description: "Open a dispute for a transaction you want to contest."
---

## Endpoint

```
POST /clients/business/disputes
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your public API key.
</ParamField>

<ParamField header="x-api-secret" type="string" required>
  Your API secret. Use a `stl_test_...` value in sandbox, `stl_live_...` in production.
</ParamField>

<ParamField header="businessId" type="string" required>
  Your business ID.
</ParamField>

## Request body

<ParamField body="transactionId" type="string" required>
  The ID of the transaction you are disputing.
</ParamField>

<ParamField body="description" type="string" required>
  A clear explanation of the issue with the transaction.
</ParamField>

## Response

<ResponseField name="status" type="boolean">
  `true` on success.
</ResponseField>

<ResponseField name="message" type="string">
  Confirmation message, e.g. `"Dispute created successfully"`.
</ResponseField>
