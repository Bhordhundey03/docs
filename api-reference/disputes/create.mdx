---
title: "Create dispute"
description: "Open a dispute for a transaction you want to contest."
---

## Endpoint

```
POST /clients/business/disputes
```

## Headers

<ParamField header="Authorization" type="string" required>
  `Bearer YOUR_ACCESS_TOKEN`
</ParamField>

<ParamField header="SECRET_KEY" type="string" required>
  Your API secret key.
</ParamField>

<ParamField header="businessId" type="string" required>
  Your business ID.
</ParamField>

## Request body

<ParamField body="transactionId" type="string" required>
  The ID of the transaction you are disputing.
</ParamField>

<ParamField body="description" type="string" required>
  A clear explanation of the issue with the transaction.
</ParamField>

## Response

<ResponseField name="status" type="boolean">
  `true` on success.
</ResponseField>

<ResponseField name="message" type="string">
  Confirmation message, e.g. `"Dispute created successfully"`.
</ResponseField>

## Code examples

<CodeGroup>
```bash cURL
curl -X POST https://sandbox.stellasbank.com/api/v1/clients/business/disputes \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "SECRET_KEY: YOUR_SECRET_KEY" \
  -H "businessId: YOUR_BUSINESS_ID" \
  -H "Content-Type: application/json" \
  -d '{
    "transactionId": "txn_abc001",
    "description": "Amount charged does not match the agreed amount for this transaction."
  }'
```

```js Node.js
const res = await fetch(
  "https://sandbox.stellasbank.com/api/v1/clients/business/disputes",
  {
    method: "POST",
    headers: {
      Authorization: "Bearer YOUR_ACCESS_TOKEN",
      SECRET_KEY: "YOUR_SECRET_KEY",
      businessId: "YOUR_BUSINESS_ID",
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      transactionId: "txn_abc001",
      description: "Amount charged does not match the agreed amount.",
    }),
  }
);
const json = await res.json();
```

```python Python
import requests

res = requests.post(
    "https://sandbox.stellasbank.com/api/v1/clients/business/disputes",
    headers={
        "Authorization": "Bearer YOUR_ACCESS_TOKEN",
        "SECRET_KEY": "YOUR_SECRET_KEY",
        "businessId": "YOUR_BUSINESS_ID",
    },
    json={
        "transactionId": "txn_abc001",
        "description": "Amount charged does not match the agreed amount.",
    },
)
print(res.json())
```
