---
title: "List member roles"
description: "Retrieve the available roles that can be assigned when inviting a new member."
---

## Endpoint

```
GET /clients/members/roles
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your public API key.
</ParamField>

<ParamField header="x-api-secret" type="string" required>
  Your API secret. Use a `stl_test_...` value in sandbox, `stl_live_...` in production.
</ParamField>

<ParamField header="businessId" type="string" required>
  Your business ID.
</ParamField>

## Response

<ResponseField name="status" type="boolean">
  `true` on success.
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable outcome.
</ResponseField>

<ResponseField name="data" type="array">
  Array of role objects.
</ResponseField>

<ResponseField name="data[].id" type="string">
  Unique role ID. Pass this as `memberRoleId` when inviting a member.
</ResponseField>

<ResponseField name="data[].name" type="string">
  Human-readable role name, e.g. `"Admin"` or `"Viewer"`.
</ResponseField>
