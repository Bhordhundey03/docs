title: "Configure billers"
description: "Update the set of billers (bank codes) enabled for your account."
---

## Endpoint

```
POST /clients/settings/configure-billers
```

## Headers

<ParamField header="Authorization" type="string" required>
  `Bearer YOUR_ACCESS_TOKEN`
</ParamField>

<ParamField header="SECRET_KEY" type="string" required>
  Your API secret key.
</ParamField>

<ParamField header="businessId" type="string" required>
  Your business ID.
</ParamField>

## Code examples

<CodeGroup>
```bash cURL
curl -X POST https://sandbox.stellasbank.com/api/v1/clients/settings/configure-billers \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "SECRET_KEY: YOUR_SECRET_KEY" \
  -H "businessId: YOUR_BUSINESS_ID" \
  -H "Content-Type: application/json" \
  -d '{
    "billers": ["044", "058", "011"]
  }'
```

```js Node.js
const res = await fetch(
  "https://sandbox.stellasbank.com/api/v1/clients/settings/configure-billers",
  {
    method: "POST",
    headers: {
      Authorization: "Bearer YOUR_ACCESS_TOKEN",
      SECRET_KEY: "YOUR_SECRET_KEY",
      businessId: "YOUR_BUSINESS_ID",
      "Content-Type": "application/json",
    },
    body: JSON.stringify({ billers: ["044", "058", "011"] }),
  }
);
const json = await res.json();
```

```python Python
import requests

res = requests.post(
    "https://sandbox.stellasbank.com/api/v1/clients/settings/configure-billers",
    headers={
        "Authorization": "Bearer YOUR_ACCESS_TOKEN",
        "SECRET_KEY": "YOUR_SECRET_KEY",
        "businessId": "YOUR_BUSINESS_ID",
    },
    json={"billers": ["044", "058", "011"]},
)
print(res.json())
```
